WutheringWaves Character Selection is an interactive web application that allows fans of the popular game "Wuthering Waves" to create and share personalized character selection cards. This project provides a visually appealing grid interface where users can select their favorite characters for various categories such as "Favorite Character," "Best Story," and "Most Influential."

鸣潮角色选择是一个交互式网络应用程序，允许《鸣潮》的粉丝创建和分享个性化的角色选择卡。该项目提供了一个视觉吸引人的网格界面，用户可以在其中为“最喜欢的角色”、“最佳故事”和“最有影响力”等各个类别选择他们喜欢的角色。

Key Features:  
主要功能：
- Interactive Grid Interface: A 3x4 grid with 11 categories for character selection
- 交互式网格界面：一个 3x4 的网格，包含 11 个角色选择类别

- Image Generation: Converts user selections into a shareable image using html2canvas
- 图像生成：将用户选择转换为可分享的图像，使用 html2canvas 技术

- QR Code Integration: Includes a QR code linking to the GitHub repository
- 二维码集成：包含一个链接到 GitHub 仓库的二维码

- Responsive Design: Works seamlessly on both desktop and mobile devices
- 响应式设计：在桌面和移动设备上均能无缝运行

This project is built with pure HTML, CSS, and JavaScript, making it lightweight and easy to deploy. It's designed to be hosted on GitHub Pages, providing an accessible platform for Wuthering Waves fans to express their preferences and share them with the community.

该项目使用纯 HTML、CSS 和 JavaScript 构建，使其轻量且易于部署。它设计为托管在 GitHub Pages 上，为《鸣潮》的粉丝提供一个可访问的平台，让他们表达自己的偏好并与社区分享。
